# Guia para criar a primeira campanha

1. Entre no painel como Administrador.
2. Abra `Campanhas`.
3. Clique em `Nova campanha`.
4. Defina nome: `Revisao Premiada`.
5. Configure periodo, horarios e status ativo.
6. Escolha campos obrigatorios: nome, telefone, WhatsApp, placa, marca, modelo e KM.
7. Ative `1 giro por placa` e `1 giro por telefone`.
8. Adicione regras de participacao.
9. Configure link do Grupo VIP.
10. Gere QR Code e link de compartilhamento.
11. Teste com um telefone e placa reais antes de divulgar.
