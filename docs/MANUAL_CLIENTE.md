# Manual do Cliente

## Como participar

1. Acesse o link da campanha pelo Instagram, WhatsApp, QR Code, Facebook ou Google.
2. Preencha seus dados e dados do veiculo.
3. Aceite termos, politica de privacidade, regulamento, contato via WhatsApp e confirmacao de dados verdadeiros.
4. Confira a placa antes de girar.
5. Gire a roleta.

## Como resgatar

1. Guarde o codigo do premio.
2. Apresente o comprovante digital na David Auto Center.
3. A placa cadastrada deve corresponder ao veiculo presente.
4. O premio so vale se estiver registrado no sistema com codigo DAC e QR Code.

## Indicacao

Use seu link exclusivo para convidar amigos. Indicacoes podem contar para ranking, niveis VIP, pontos e campanhas futuras.

## Acesso do cliente

O cliente nao precisa de login nem senha. O comprovante aparece logo apos o giro e pode ser aberto pelo codigo/QR Code.
