# Manual do Atendente

## Consultar cliente

1. Acesse `/admin`.
2. Abra `Clientes`.
3. Pesquise por nome, telefone, placa, cidade, veiculo, nivel ou campanha.

## Validar premio

1. Abra `Resgates`.
2. Digite o codigo `DAC-ANO-CODIGO`.
3. Confira cliente, telefone, placa, premio, campanha, validade e status.
4. Se o veiculo estiver presente e a placa corresponder, marque como resgatado.

## Corrigir dados

Atendentes devem solicitar permissao de gerente ou administrador para corrigir placa, telefone, nome ou veiculo. Toda alteracao precisa de motivo e log.

## WhatsApp

Use os botoes de WhatsApp do painel para confirmar agendamento, avisar sobre premio pendente ou chamar clientes sem retorno.
