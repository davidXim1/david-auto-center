# Guia para criar premios

## Campos obrigatorios

- Nome
- Descricao
- Categoria
- Valor estimado
- Quantidade disponivel
- Probabilidade
- Validade
- Status

## Categorias

- Comum
- Intermediario
- Premium
- Especial
- Personalizado

## Regras recomendadas

- Premios premium devem ter probabilidade menor e limite por tempo.
- Todo participante deve ganhar algo.
- Quando estoque chegar a zero, o premio deve ficar esgotado.
- Use observacoes para regras como agendamento, compatibilidade e nao cumulatividade.
